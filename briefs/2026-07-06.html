TL;DR

  • Claude Sonnet 5 devient le défaut de Claude Code (30 juin, v2.1.197). 85.2 % SWE-bench Verified, 63.2 % Pro, contexte 1M natif, thinking adaptatif jusqu'à xhigh. Pricing d'introduction $2/$10 par M tokens jusqu'au 31 août (puis $3/$15) — ~40 % du tarif Opus. Il resserre l'écart agentique avec Opus 4.8 (Terminal-Bench 2.1 à 80.4 %, OSWorld 81.2 %) en jouant le rapport perf/prix plutôt que la performance brute face à GPT-5.5 (88.7 % Verified).
  • Claude Code enchaîne six versions + Fable 5 est déverrouillé. 2.1.196 → 2.1.201 : subagents en arrière-plan par défaut (auto-commit/push/PR draft), Claude in Chrome en GA, permission « Manual » par défaut, modèles par défaut d'organisation, skill /dataviz. En parallèle, les export controls sur Fable 5 (95 % Verified) et Mythos 5 sont levés le 30 juin ; Fable 5 revient mondialement le 1er juillet, quotas bridés à 50 % jusqu'au 7.
  • MCP se généralise et l'open weights chinois pousse. Google met le MCP natif dans Gemini Spark sur macOS (agent desktop, AI Ultra 99 $/mois) ; xAI ouvre Voice Agent Builder (voix single-stack, MCP natif, ~$0.06/min). Côté open weights : Meituan open-source LongCat-2.0 (1.6T MoE MIT, entraîné 100 % sur ASIC chinois, SWE-bench Pro 59.5 self-reported) et Z.ai lance ZCode autour de GLM-5.2 (18-160 $/mois).

🎯 Implications Lynxter

Sonnet 5 est déjà le modèle que tes sessions Claude Code utilisent — c'est l'action n° 1 de la semaine. Dès la v2.1.197, le défaut n'est plus Opus mais Sonnet 5 : pour tes routines (brief veille, scan tickets, audit de profils S300X/S600D), ça veut dire un coût divisé par ~2,5 pendant la promo $2/$10, contre un léger recul sur les tâches multi-fichiers mal spécifiées (63.2 % SWE-bench Pro vs 69.2 % pour Opus 4.8). Ne subis pas le défaut : relance une routine représentative sur Sonnet 5, compare à ta baseline Opus, et pose une règle explicite — « Opus (ou Fable 5) pour les audits firmware, Sonnet 5 pour le volume » — puis versionne-la dans les settings d'équipe via les modèles par défaut d'organisation (arrivés en 2.1.196). La fenêtre promo court jusqu'au 31 août : si un engagement annuel Cowork/Claude Code est sur la table, c'est le moment de figer le tarif avant le passage à $3/$15.

La vague Claude Code change ta boucle agentique, pas seulement tes prix. Les subagents tournent désormais en arrière-plan par défaut (2.1.198) et, lancés via claude agents, commitent/poussent/ouvrent une PR draft en fin de tâche : les jobs longs (génération de profils d'impression, audit croisé de paramètres firmware, balayage de tickets) passent d'un mode bloquant à du fire-and-forget avec livrable Git. Côté gouvernance, le mode permission « Manual » par défaut (2.1.200) redonne un cran de contrôle sur les actions sensibles — à ajuster par workflow pour ne pas ralentir les boucles — et Claude in Chrome passe en GA, brique réelle pour piloter les vieux outils web sans API. Structure dès cette semaine tes tâches longues en sous-agents nommés et décide explicitement où tu gardes Manual vs auto-mode.

Fable 5 est de retour, mais l'épisode est un rappel sur la dépendance à un modèle unique. Le modèle le plus capable d'Anthropic (95 % SWE-bench Verified) est de nouveau accessible dans Claude Code après 19 jours de blackout purement réglementaire — mais bridé à 50 % des quotas hebdo jusqu'au 7 juillet, puis via crédits. Si un workflow Lynxter en dépend, planifie les runs lourds cette semaine et garde Opus 4.8 / Sonnet 5 en repli. Le vrai enseignement dépasse Fable 5 : un blackout d'origine réglementaire peut sortir un modèle du marché du jour au lendemain, indépendamment de sa qualité. Garde tes routines de production portables entre modèles (Sonnet 5 / Opus 4.8 / Fable 5), avec un fallback testé — et vois la standardisation de MCP (adopté cette semaine par Google et xAI) comme un atout : les serveurs MCP que tu construis pour tes outils métier deviennent réutilisables au-delà d'un seul fournisseur.

Claude Sonnet 5 — 85.2 % SWE-bench Verified, contexte 1M natif, promo $2/$10 et défaut de Claude Code

🎯 Lynxter 2026-06-30

Anthropic sort Claude Sonnet 5 le 30 juin et en fait le modèle par défaut de Claude Code dès la v2.1.197. Il monte à 85.2 % sur SWE-bench Verified et 63.2 % sur SWE-bench Pro, avec un contexte natif de 1M tokens et un thinking adaptatif jusqu'à l'effort xhigh. Positionné comme le Sonnet le plus agentique, il resserre l'écart avec Opus 4.8 (88.6 % Verified) sur le tool use et le computer use — Terminal-Bench 2.1 à 80.4 %, OSWorld-Verified 81.2 % — tout en restant nettement moins cher : pricing d'introduction $2/$10 par M tokens jusqu'au 31 août, puis $3/$15. Face à GPT-5.5 (88.7 % Verified) il ne joue pas la performance brute mais le rapport perf/prix sur les boucles agentiques longues.

Claude Code 2.1.196 → 2.1.201 — subagents en arrière-plan par défaut, Claude in Chrome en GA, permission « Manual » par défaut

🎯 Lynxter 2026-07-03

Six versions de Claude Code livrées dans la fenêtre (2.1.196 le 29 juin → 2.1.201 le 3 juillet). Trois changements structurants : les subagents tournent en arrière-plan par défaut et notifient à la fin (2.1.198), Claude in Chrome passe en disponibilité générale (2.1.198), et le mode de permission par défaut bascule sur « Manual » à travers CLI/VS Code/JetBrains (2.1.200). Les admins peuvent désormais définir des modèles par défaut d'organisation (2.1.196), les agents en arrière-plan lancés via claude agents commitent, poussent et ouvrent une PR draft à la fin d'un travail en worktree, et la skill /dataviz fait son entrée. Gros lot de correctifs fiabilité : subagents coupés par rate-limit renvoyant leur travail partiel, sessions background après sleep/wake, daemon Linux.

Fable 5 déverrouillé — les export controls levés, fin d'un blackout de 19 jours, quotas bridés jusqu'au 7 juillet

🛠 Utile 2026-07-01

Le Department of Commerce a levé le 30 juin les contrôles à l'export imposés sur Fable 5 et Mythos 5, mettant fin à un blackout mondial de 19 jours (restriction déclenchée après qu'une équipe Amazon a décrit un jailbreak contournant une sauvegarde cyber de Fable 5). Fable 5 a été redéployé mondialement dès le 1er juillet sur Claude Platform, Claude.ai, Claude Code et Claude Cowork ; il est inclus jusqu'à 50 % des limites hebdomadaires jusqu'au 7 juillet pour Pro/Max/Team et enterprise sélectionnés, puis via crédits. Mythos 5, lui, n'est réintroduit qu'aux organisations US approuvées via Project Glasswing. Anthropic a publié le 2 juillet un brouillon de framework de sévérité des jailbreaks avec ses partenaires Glasswing (Amazon, Microsoft, Google).

anthropic.com/news/redeploying-fable-5 · cnbc.com · forbes.com

Claude Science — workbench agentique pour la recherche (60+ bases scientifiques, sur Opus 4.8)

· Info 2026-06-30

Anthropic lance le 30 juin Claude Science, présenté comme l'équivalent de Claude Code pour la recherche : un agent coordinateur sur Opus 4.8 avec 60+ skills et connecteurs préconfigurés (génomique, protéomique, biologie structurale, cheminformatique), en beta immédiate pour Pro/Max/Team/Enterprise. Sans lien direct impression 3D, mais signal net — Anthropic généralise le pattern « agent coordinateur + skills métier » à de nouveaux verticaux.

· Info 2026-07-05

OpenAI a poussé la préversion 0.143 du Codex CLI entre le 1er et le 5 juillet (alpha.32 → alpha.36), la 0.142.5 stable du 1er juillet restant un simple patch de sécurité (fuite de payloads WebSocket dans les trace logs corrigée). Gemini Spark débarque sur macOS — agent desktop autonome avec MCP natif et automatisation de fichiers locaux

🛠 Utile 2026-07-01

Google étend son agent personnel Gemini Spark à l'app desktop macOS (beta, abonnés AI Ultra à 99 $/mois, US uniquement) le 1er juillet. Spark sort de la fenêtre de chat pour exécuter des tâches sur les fichiers et apps locaux : tri automatique des PDF d'un dossier Downloads en sous-dossiers labellisés, extraction de chiffres depuis des factures locales vers un tableur Workspace. Positionnement frontal face à Claude computer use sur le terrain de l'agent desktop, mais verrouillé au tier le plus cher.

Z.ai lance ZCode — environnement de dev agentique (Electron) autour de GLM-5.2, de 18 à 160 $/mois

🛠 Utile 2026-07-02

Z.ai (ex-Zhipu AI) a lancé ZCode le 2 juillet, un environnement de développement agentique (app Electron, macOS/Windows/Linux) bâti autour de GLM-5.2. L'UI met la conversation agent au centre, entourée d'un file manager, terminal, panneau Git et preview navigateur live — positionnement frontal contre Cursor, Claude Code, GitHub Copilot et Google Antigravity. Essai gratuit 5 jours (5M tokens/jour), puis GLM Coding Plan de 18 $ (Lite) à 160 $/mois (Max), avec support BYOK pour modèles tiers. GLM-5.2 (contexte 1M) affiche 62.1 sur SWE-bench Pro (vs 58.6 pour GPT-5.5, 69.2 pour Opus 4.8). Point de vigilance : chaque appel API GLM-5.2 est soumis au droit chinois.

zcode.z.ai/en/docs/welcome · venturebeat.com

Meituan open-source LongCat-2.0 — 1.6T MoE sous MIT, agentic coding, entraîné 100 % sur puces chinoises

🛠 Utile 2026-06-30

Meituan a open-sourcé le 30 juin LongCat-2.0, un MoE de 1.6T paramètres à contexte natif 1M tokens sous licence MIT, taillé pour le coding agentique. Le modèle tournait depuis ~2 mois sur OpenRouter sous le codename anonyme « Owl Alpha », devenant l'un des plus utilisés de la plateforme (~10.1T tokens/mois) avant la révélation. Meituan revendique 59.5 sur SWE-bench Pro (self-reported) contre 58.6 pour GPT-5.5 — marge sous 1 point, chiffres non vérifiés indépendamment, donc compétitif plutôt que supérieur. Fait notable : premier modèle trillion-paramètres entraîné et servi entièrement sur 50 000+ ASIC chinois, sans aucun GPU Nvidia.

longcatai.org/models/longcat-2 · venturebeat.com

xAI lance Voice Agent Builder (beta) — plateforme voix single-stack à $0.06/min avec MCP natif

🛠 Utile 2026-07-01

xAI a lancé le 1er juillet Voice Agent Builder en beta, une plateforme no-code qui transforme une description en langage naturel en agent vocal téléphonique live en ~2 minutes. Le stack packagé fournit téléphonie, retrieval, tools, guardrails et observabilité, avec un point notable : support MCP natif out of the box (branchement de serveurs MCP existants, WebSocket/SIP). Le chemin speech-to-speech est tightly coupled à Grok Voice, pas trois briques assemblées. Pricing agressif : $0.05/min audio + $0.01/min téléphonie (~$0.06/min tout compris), avec 80+ voix ou voice cloning depuis ~2 min d'audio. Face aux stacks OpenAI Realtime et aux assemblages multi-fournisseurs.

x.ai/news — Voice Agent Builder · cryptobriefing.com

Cursor lance son app iOS (beta publique) — piloter des agents de code depuis le mobile

· Info 2026-06-29

Cursor a publié le 29 juin sa première app native iPhone/iPad (beta publique, tous plans payants), dans la foulée de son rachat par SpaceX. L'app lance des always-on agents dans le cloud ou télécommande des agents tournant sur le poste de bureau, avec contrôle vocal, notifications push et review/merge de PR depuis le mobile (promo de lancement -75 % sur Composer 2.5 jusqu'au 5 juillet).
