TL;DR

  • Agent View + /goal dans Claude Code 2.1 : une vue CLI unifiée pour toutes tes sessions parallèles, et une commande qui fait tourner Claude en autonomie jusqu'à un objectif défini — exactement le pattern des skills long-running Lynxter.
  • Anthropic ouvre deux suites verticales MCP en deux jours : Legal (20+ connecteurs, 12 plugins pratiques) et Small Business (QuickBooks, HubSpot, Canva) — signal clair de la stratégie "Claude everywhere" sur tous les marchés.
  • OpenAI passe la Realtime API en GA avec gpt-realtime, MCP distant, inputs image et appels SIP — la voix dans les agents de support devient plomberie standard, pas R&D.

🎯 Implications Lynxter

Agent View + /goal transforment ton workflow Cowork dès maintenant. Tu peux lancer le brief veille-IA en background (claude --bg "génère le brief semaine"), le scan tickets support en background, et garder une session interactive pour le dev en cours — le tout dans une seule vue CLI. Le /goal est fait pour tes skills long-running : au lieu de superviser chaque étape, tu définis un objectif ("pusher le brief complet", "fermer tous les tickets P1") et Claude tourne en autonomie. C'est la pièce manquante entre Routines et les sessions interactives.

Regarde les connecteurs Claude for Small Business. Si Lynxter gère des devis, des commandes S300X/S600D ou du CRM via HubSpot, les connecteurs Anthropic sont disponibles dans le plan Claude dès maintenant. Une intégration HubSpot → Claude → workflow devis automatisé peut se tester en quelques heures — sans passer par une intégration custom. Le connesteur QuickBooks est intéressant si la comptabilité passe par là aussi.

Marque la Realtime API OpenAI dans ton radar support vocal. gpt-realtime avec support MCP + appels SIP ouvre une voie pour un agent vocal qui répond aux appels clients entrants (déballage, mise en route S300X), accède à l'historique tickets via MCP, et triage en autonomie. Ce n'est pas un chantier pour cette semaine, mais le benchmark est maintenant possible — OpenAI en GA pricing public, Voxtral/Mistral en option on-prem pour le français.

Synthèse

Semaine dense côté Anthropic sur deux fronts distincts : l'outillage dev avec Claude Code 2.1 (Agent View, /goal, plugin diagnostics) et les suites verticales MCP (Legal + Small Business), qui montrent une stratégie d'expansion tous azimuts au-delà du dev. OpenAI accélère sur la voix avec gpt-realtime en GA — MCP support et appels SIP dans la même release — et étend Codex au mobile en preview. Google fait du pre-heating avant son I/O du 19 : Gemini Intelligence sur Android annoncé lors de l'Android Show, les vraies annonces Gemini arrivent mardi. Meta et Mistral sont absents cette semaine. Côté acteurs secondaires : Perplexity rend Computer disponible sur Mac et Comet sur iOS — avec Claude Opus 4.7 comme orchestrateur par défaut. xAI lance Grok Build, agent CLI de coding (concurrent direct à Claude Code et Codex), en beta SuperGrok Heavy. Cursor sort Composer 2, modèle coding frontier-level à $0.50/M input, avec agents parallèles et intégration Microsoft Teams.

gpt-realtime + Realtime API en GA — voix, MCP, appels SIP

🛠 Utile 2026-05-14

OpenAI passe la Realtime API en général availability avec gpt-realtime, son modèle speech-to-speech le plus avancé. Trois nouvelles capacités clés : support MCP distant (connecteurs de données tiers dans les sessions vocales), inputs image en temps réel (l'agent voit ce que l'utilisateur montre), appels SIP (l'API peut recevoir des appels téléphoniques VoIP entrants). Les "reusable prompts" permettent de sauvegarder la config agent (prompt système + tools + variables) pour la réutiliser cross-sessions.

openai.com/index/advancing-voice-intelligence-with-new-models-in-the-api

Codex from anywhere — dev sur mobile en preview

🛠 Utile 2026-05-14

Codex arrive dans l'app ChatGPT iOS (preview). L'app se connecte à la machine où Codex tourne (laptop, Mac mini, managed remote) et charge l'état live de tous les threads. Depuis le téléphone : review des outputs (diffs, screenshots, résultats tests), approbation des commandes, changement de modèle, démarrage de nouveaux threads. Les fichiers et credentials restent sur la machine distante. Concurrent direct aux Remote Control sessions de Claude Code — même paradigme, écosystème différent.
